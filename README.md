# Website Polling & Aspirasi Warga Kota Magelang

## Deskripsi
Website untuk menampung aspirasi dan polling pendapat warga terhadap kebijakan Kota Magelang.

## Fitur
- Verifikasi NIK (16 digit)
- Setiap NIK maksimal 10x penggunaan per tahun
- Polling untuk 5 kebijakan
- Aspirasi umum
- Penyimpanan data di GitHub

## Cara Penggunaan
1. Masukkan NIK (contoh: 1111111111111111)
2. Pilih Polling atau Aspirasi
3. Isi form yang tersedia
4. Submit

## Teknologi
- HTML5
- CSS3
- JavaScript
- GitHub API (sebagai database)

## Cara Deploy ke GitHub Pages
1. Upload semua file ke repository GitHub
2. Aktifkan GitHub Pages di Settings
3. Akses website di: https://anaksubuh.github.io/KETARA.github.io/

## Catatan Keamanan
⚠️ Jangan menyimpan token GitHub di file yang di-upload ke repository publik!
